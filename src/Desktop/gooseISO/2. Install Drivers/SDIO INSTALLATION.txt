If you are familiar with SDIO, you do not need to read this.

1. Install SDIO using the script provided
2. Accept the Terms
3. Click "Download Indexes Only"
4. Wait for Indexes to download
5. Tick the Devices you'd like to update drivers for. Remember, less is usually better so don't download them all unless you need to
6. Done!

If you're not sure what to install, read below.

Minimal:
- Networking (will lose network tweaks, check the troubleshooting folder to redo them!)
- USB (if needed)
- Graphics (from your vendor's website, not SDIO)

Recommended:
- Networking (will lose network tweaks, check the troubleshooting folder to redo them!)
- USB (if needed)
- Graphics (from your vendor's website, not SDIO)
- Audio (if needed)
- Bluetooth (if needed)
